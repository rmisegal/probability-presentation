What Probability Theory Is

מה היא תורת ההסתברות

הגדרה ומושגי יסוד

תורת ההסתברות היא ענף במתמטיקה העוסק במדידה וניתוח של אי-ודאות. היא מספקת לנו כלים מתמטיים לכמת את הסיכוי שאירוע מסוים יתרחש.

תורת ההסתברות חיונית בתחומים רבים:

  • למידת מכונה ובינה מלאכותית
  • סטטיסטיקה וניתוח נתונים
  • פיזיקה ומדעי הטבע
  • כלכלה ומימון
  • רפואה ואפידמיולוגיה
דוגמאות מהחיים האמיתיים:
• מה הסיכוי שיירד גשם מחר?
• מה ההסתברות לזכות בלוטו?
• מה הסיכוי שמטופל יחלים מטיפול מסוים?

Python Code Example

slide1.py - Basic Probability Concepts
import random import numpy as np import matplotlib.pyplot as plt def basic_probability_demo(): """ הדגמה בסיסית של מושגי הסתברות Basic demonstration of probability concepts """ print("=== תורת ההסתברות - דוגמאות בסיסיות ===") print("=== Probability Theory - Basic Examples ===") # הטלת מטבע - Coin flip print("\n1. הטלת מטבע (Coin Flip):") coin_results = [] for i in range(1000): result = random.choice(['ראש', 'עץ']) # Heads, Tails coin_results.append(result) heads_count = coin_results.count('ראש') probability_heads = heads_count / 1000 print(f"מספר 'ראש' ב-1000 הטלות: {heads_count}") print(f"הסתברות ניסיונית ל'ראש': {probability_heads:.3f}") print(f"הסתברות תיאורטית ל'ראש': 0.500") # הטלת קובייה - Dice roll print("\n2. הטלת קובייה (Dice Roll):") dice_results = [random.randint(1, 6) for _ in range(1000)] for number in range(1, 7): count = dice_results.count(number) probability = count / 1000 print(f"מספר {number}: {count} פעמים, הסתברות: {probability:.3f}") return coin_results, dice_results # הרצת הדוגמה if __name__ == "__main__": coin_data, dice_data = basic_probability_demo()

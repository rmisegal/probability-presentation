Multiple Independent Observations

תצפיות עצמאיות מרובות

עצמאות בהסתברות

תצפיות עצמאיות: כאשר תוצאת ניסוי אחד אינה משפיעה על תוצאת ניסוי אחר.

דוגמאות לעצמאות:

  • הטלות מטבע רצופות
  • הטלות קובייה רצופות
  • משיכת קלפים עם החזרה

דוגמאות לאי-עצמאות:

  • משיכת קלפים ללא החזרה
  • בחירת כדורים מקופסה ללא החזרה
P(A and B) = P(A) × P(B)
עבור אירועים עצמאיים: הסתברות שני אירועים = מכפלת ההסתברויות
דוגמאות חישוב:
• P(HH) = P(H) × P(H) = 1/2 × 1/2 = 1/4
• P(HHH) = 1/2 × 1/2 × 1/2 = 1/8
• P(exactly 2H in 3 tosses) = 3/8
• P(at least 2H in 3 tosses) = 4/8 = 1/2

import itertools from fractions import Fraction def multiple_coin_flips(n): """חישוב כל התוצאות האפשריות ל-n הטלות""" outcomes = list(itertools.product(['H', 'T'], repeat=n)) print(f"הטלת {n} מטבעות:") print(f"מספר תוצאות אפשריות: {len(outcomes)}") print(f"תוצאות: {outcomes}") # ספירת ראשים בכל תוצאה heads_count = {} for outcome in outcomes: h_count = outcome.count('H') if h_count not in heads_count: heads_count[h_count] = 0 heads_count[h_count] += 1 print(f"\nהתפלגות מספר ראשים:") for heads, count in sorted(heads_count.items()): prob = count / len(outcomes) print(f"{heads} ראשים: {count} דרכים, P = {prob:.3f}") def consecutive_heads_probability(): """הסתברות לרצפי ראשים""" print("\n=== הסתברות רצפי ראשים ===") for n in range(1, 6): prob = (1/2) ** n print(f"P({'H' * n}) = (1/2)^{n} = {prob:.4f}") # דוגמה לשימוש multiple_coin_flips(3) consecutive_heads_probability()
P(exactly k successes in n trials) = C(n,k) × p^k × (1-p)^(n-k)
התפלגות בינומית: הסתברות לקבל בדיוק k הצלחות ב-n ניסויים
מרחב מדגם ל-3 הטלות:
Ω = {HHH, HHT, HTH, HTT, THH, THT, TTH, TTT}
|Ω| = 2³ = 8

0 heads: {TTT} → 1 way → P = 1/8
1 head: {HTT, THT, TTH} → 3 ways → P = 3/8
2 heads: {HHT, HTH, THH} → 3 ways → P = 3/8
3 heads: {HHH} → 1 way → P = 1/8
